# vignac

Perl script, simplified version of whois. Returning results without unnecessary lines.

### Prerequisites

```
apt-get install perl libwww-perl libwww-mechanize-perl libjson-perl
```
### Usage example:

```
perl vignac.pl yahoo.com
```
